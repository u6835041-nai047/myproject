# My First Web Project

This is a simple Hello World web project created while learning
basic web development and Git.

## Files in this project
- `index.html` – Main HTML file
- `bluestyle.css` – Basic styling for the page
- `README.md` – Project description

## Purpose
The purpose of this project is to:
- Practice using Git and GitHub
- Learn basic HTML and CSS
- Understand how files are tracked and committed

## Status
This project is a beginner exercise and may be updated in the future.
